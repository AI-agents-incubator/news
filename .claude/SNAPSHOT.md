# Project Snapshot

## Текущее состояние (2026-04-13)

### Что работает
- **Сбор статей**: Telegram бот (@aivideoproducer_bot) принимает URL от пользователя
- **Обогащение контента**: local-fetcher.js на Mac открывает URL в Chrome, извлекает текст
- **Генерация дайджестов**: Claude API (Opus 4.6), 2-фазный процесс (комментарии + сборка)
- **Dashboard**: news.questtales.com — табличка с дайджестами, превью, копирование, публикация
- **Публикация в Telegram**: Bot API в канал @alexkrol, разбивка длинных сообщений
- **Публикация на Facebook Page**: Graph API, страница Alex Krol
- **Публикация в Facebook Profile**: Patchright (stealth Playwright), тестовый аккаунт
- **Автоматический watcher**: launchd на Mac, каждые 5 мин проверяет новые дайджесты
